## 🎨 **Key Visual Elements**

### **Colors Used:**
- **Rider Section:** Emerald Green (#10B981) - Symbolizes earning, growth
- **User Section:** Blue (#3B82F6) - Symbolizes trust, ease
- **Why Bihar Section:** Green Gradient - Strong, powerful
- **CTAs:** Mixed (Emerald, Blue, Orange, Gradient) - Action-focused
- **Dark Mode:** All colors adapt to dark theme

### **Typography:**
- **Hero Headline:** 7XL, Black, Bold (Most attention-grabbing)
- **Section Titles:** 5-6XL, Black, Bold
- **Card Titles:** XL-2XL, Bold
- **Body Text:** Base, Regular (Easy to read)

### **Animations:**
- Fade-in on scroll for each section
- Card hover: lift up (+10px), shadow increase
- Button click: scale 0.95 (satisfying click feedback)
- Smooth transitions between sections
- Scroll-triggered stagger animations

## 📱 **How It Works on Different Devices**

### **Desktop (1440px+)**
- 3-4 column grids for cards
- Side-by-side Instant vs Reserve
- Full width navigation
- Large typography
- Best readability

### **Tablet (768px - 1024px)**
- 2 column grids
- Adjusted font sizes
- Full-width cards
- Touch-friendly buttons

### **Mobile (320px - 640px)**
- Single column layout
- 56px button height (finger-friendly)
- Large font sizes
- Hamburger menu
- Optimized for low-bandwidth

## 🌙 **Dark Mode**

Click the moon icon (🌙) in navbar to toggle dark mode:
- **Light:** White backgrounds, dark text
- **Dark:** Dark gray backgrounds, light text
- **Colors:** Adapt for visibility (e.g., Emerald 400 instead of 600)
- **Persists:** Remembered in localStorage

## 🚀 **How to View**

### **Option 1: Local Development**
```bash
cd frontend
npm run dev
# Visit http://localhost:5174
```

### **Option 2: Production Build**
```bash
npm run build
npm run preview
# Visit http://localhost:4173
```

### **Option 3: On Mobile**
If running on your machine, access from mobile:
- Same WiFi as your computer
- Get your machine IP: `ipconfig` (Windows) or `ifconfig` (Mac/Linux)
- Visit: `http://[YOUR_IP]:5174` from mobile
